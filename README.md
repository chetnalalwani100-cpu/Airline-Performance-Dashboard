# Airline Performance Dashboard

## Overview

This project presents a comprehensive Airline Performance Analytics Dashboard developed in Power BI to analyze flight delays, weather impact, airport performance, route efficiency, and operational trends.

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## Key Features

### Executive Operations Summary
- Airport taxi-out performance analysis.
- Route and weather hierarchy analysis.
- Climate pattern monitoring across airports.
- Weather-wise delay trend comparison.

### Flight Delay Departure Analysis
- Departure delay category distribution.
- Distance vs departure delay analysis.
- Weather condition and temperature impact analysis.
- Aircraft-wise departure delay frequency tracking.

### Flight Delay Root Cause Analysis
- Delay distribution by temperature.
- Route distance vs delay performance.
- Flight schedule and delay monitoring.
- Weekend delay flow analysis.

### Weather Impact Intelligence
- Weather-driven delay intelligence reporting.
- Temperature and climate pattern analysis.
- Airport weather distribution insights.
- Weather condition influence analysis using AI visuals.

## Key Insights

- Weather conditions significantly influence flight delays.
- Airport operational efficiency varies across routes and destinations.
- Temperature, humidity, and wind speed impact departure performance.
- Certain airports consistently experience higher delay frequencies.
- Route distance plays a significant role in delay patterns.

## Project Outcome

The dashboard helps airlines and airport authorities monitor operational performance, identify delay root causes, analyze weather impacts, and support data-driven decision-making to improve efficiency and customer experience.
